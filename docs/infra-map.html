Dedup Mechanism

integration-state-$$.json (session-scoped) tracks what has fired. Pipeline gates read it before acting — if a skill already handled the concern, the gate skips. Reset at pipeline completion by close-loop gate. Separate from /tmp/pipeline-state.json (ephemeral display for statusline). Foxhound nudge uses its own flag: /tmp/foxhound-session-$$. Eval capture logs to /tmp/eval-capture-$$.jsonl.
