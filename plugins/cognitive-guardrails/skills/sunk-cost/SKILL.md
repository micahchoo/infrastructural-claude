# Sunk Cost Check

Each step you take makes you more committed to the path — not because it's right, but because every new step was generated conditional on the previous ones. By step 15, you're locked in even if step 3 was wrong.

Summarize only: what is the current state, and what is the goal? Now forget how you got here. If you were a fresh agent encountering this state and goal for the first time, what would you do? If that answer diverges from your current plan, the divergence is evidence of sunk-cost bias. Seriously consider the fresh plan.

`[eval: sequence]` If fresh agent would diverge, stop and restart with revised approach.
