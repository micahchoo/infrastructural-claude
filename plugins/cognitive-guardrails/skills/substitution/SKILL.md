# Substitution Check

When a question is hard, you'll silently swap it for an easier one you recognize. The output feels right because it answers *a* reasonable question — just not the one that was asked.

Re-read the exact original question or subtask. Quote the specific thing being asked. Now quote the specific part of your output that addresses it. Is there a gap between what was asked and what you answered? If you addressed a more generic version, close the gap now.

`[eval: target]` Answer addresses user's actual question, not an easier substitute.
